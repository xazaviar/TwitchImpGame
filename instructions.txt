STARTING THE PROJECT
--------------------
You need two terminals open simultaneously.

  Terminal 1 - Server:
    cd packages/server
    npx tsx src/index.ts

  Terminal 2 - Client:
    cd packages/client
    npx vite

Both must be running at the same time for the game to work.


FIRST-TIME SETUP (one time only)
---------------------------------
Before the server will connect to Twitch, you need to authorize two accounts:

  1. Open http://localhost:3001/api/setup in your browser
  2. Click "Authorize Bot Account" - log in as xazaviar_bot
  3. Click "Authorize Broadcaster Account" - log in as xazaviar (your stream account)
  4. Done - tokens are saved and will auto-reconnect on every future startup.

CHAT COMMANDS
-------------
  !join        Create your imp and join the horde (first time only)
  !vote <#>    Vote during adventure/event votes (e.g. !vote 1, !vote 2)
  !stats       Check your imp's stats and current adventure info
  !loot        Check the current adventure's loot pool
  !help        List available commands


HOW THE GAME WORKS (for community)
-----------------------------------
  1. Type !join in chat to create your imp
  2. Wait for the keep countdown or the streamer starts an adventure
  3. A location vote appears - pick where the horde should go!
     - Vote "Stay at Keep" to sit this adventure out
     - Not voting = staying at the keep
     - All who vote for ANY adventure go on the winning destination
  4. The horde travels to the destination and faces 5 steps:
     Combat -> Event -> Combat -> Event -> Boss
  5. After the boss, vote to push deeper (more loot, more risk) or return home
  6. Surviving imps bring back gold and materials to the keep!

You can also vote from the web interface at http://localhost:5173/
by signing in with your Twitch account.


FOR YOUR COMMUNITY
------------------
Share the viewer URL: http://localhost:5173/
They can sign in with Twitch to vote from the web alongside chat.

(When hosted on a VPS later, this will be a public URL accessible anywhere.)


TROUBLESHOOTING
---------------
  Port 3001 already in use:
    - You likely have a previous server still running
    - Close it with Ctrl+C in its terminal, then restart

  Twitch chat not connecting:
    - Visit http://localhost:3001/api/setup and re-authorize the bot account
    - Check that your .env file has TWITCH_CLIENT_ID and TWITCH_CLIENT_SECRET

  Client won't connect to server:
    - Make sure the server terminal is running first
    - Check ports: server = 3001, client = 5173

  "No imps in the horde" when starting adventure:
    - At least one person needs to !join in chat first


PORTS SUMMARY
-------------

  5173  - Client dev server (React UI)
